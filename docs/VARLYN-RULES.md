# Varlyn Rules Reference

**System:** Varlyn D&D 5e homebrew
**Rules site:** [dnd.rigo.nu](https://dnd.rigo.nu)

This document provides a brief summary of Varlyn's key rule differences from standard D&D 5e, and links to the authoritative rules source.

> **Note:** This document is a reference summary only. For the complete and authoritative rules, always consult [dnd.rigo.nu](https://dnd.rigo.nu).

---

## What is Varlyn?

Varlyn is a homebrew D&D 5e campaign setting with customized mechanics developed for the Uppsala Rollspel group. It modifies several core D&D 5e systems to better fit the setting's themes and play style.

**Campaign history and world lore:** [dnd.rigo.nu](https://dnd.rigo.nu)

## Key Rule Differences from Standard D&D 5e

### 7 Ability Scores

Varlyn adds **Luck (LUK)** as a seventh ability score:

| Ability      | Abbreviation | Type     | Notes           |
| ------------ | ------------ | -------- | --------------- |
| Strength     | STR          | Physical | Standard        |
| Dexterity    | DEX          | Physical | Standard        |
| Constitution | CON          | Physical | Standard        |
| Intelligence | INT          | Mental   | Standard        |
| Wisdom       | WIS          | Mental   | Standard        |
| Charisma     | CHA          | Mental   | Standard        |
| Luck         | LUK          | Fortune  | Varlyn addition |

**Luck uses:**

- Spellcasting ability for certain classes (e.g., Halfling Bards)
- Luck saves
- Luck checks (fortune-based situations)
- Some folk abilities

**Reference:** [dnd.rigo.nu/rules/abilities](https://dnd.rigo.nu)

### Hierarchical Skill System

Varlyn replaces the standard flat skill list with a **skill tree system**:

#### Three Categories

1. **Untrained** — Always rollable, no cost
   - Basic skills anyone can attempt
   - No prerequisites

2. **Knowledge** — Must own to roll without disadvantage
   - Academic and lore skills
   - Cost: 1 skill point each

3. **Practical** — Hierarchical with prerequisites
   - Physical and trade skills
   - Cost: 1 skill point each
   - Must own prerequisite skill first

#### Skill Points

- Characters gain skill points each level (base rate + class bonus + folk bonus)
- Points spent to purchase skills in Knowledge and Practical trees
- Prerequisites must be met before purchasing

#### Expertise

- Expertise = **advantage** on that skill check (not double proficiency)
- Gained through class features, folk traits, or spending additional points

**Reference:** [dnd.rigo.nu/rules/skills](https://dnd.rigo.nu)

### Folk System

Varlyn's "folk" replaces the standard race system with more nuanced categories:

#### Folk Categories

| Category     | Description                            | Examples                     |
| ------------ | -------------------------------------- | ---------------------------- |
| Old          | Ancient peoples native to the world    | Dwarves, Elves               |
| Visitor      | Peoples who arrived from elsewhere     | Dragonfolk, Genasi, Tiefling |
| New          | More recently established peoples      | Humans, Halflings            |
| Awakened     | Non-humanoid creatures given sentience | Awakened (various)           |
| Else-Touched | Peoples with otherworldly influence    | Aasimar, Dhampir             |

#### 19 Folk

| Folk       | Category     | Notable Features                              |
| ---------- | ------------ | --------------------------------------------- |
| Human      | New          | Versatile, bonus skill point                  |
| Dwarf      | Old          | Subtypes: Hill, Mountain                      |
| Elf        | Old          | Subtypes: High, Wood, Dark                    |
| Halfling   | New          | Subtypes: Lightfoot, Stout; Luck spellcasting |
| Dragonfolk | Visitor      | Draconic ancestry, breath weapon              |
| Beastfolk  | New          | d30 animal type table                         |
| Birdfolk   | New          | Flight                                        |
| Lizardfolk | Visitor      | Environmental adaptations                     |
| Aasimar    | Else-Touched | Divine heritage                               |
| Genasi     | Visitor      | Elemental heritage                            |
| Tiefling   | Else-Touched | Infernal heritage                             |
| Awakened   | Awakened     | d20 creature type                             |
| Changeling | New          | Shapeshifting                                 |
| Dhampir    | Else-Touched | Vampiric traits                               |
| Floran     | New          | Plant-based life                              |
| Gnome      | Old          | Subtypes: Forest, Rock                        |
| Halforc    | New          | Orcish heritage                               |
| Selkie     | Visitor      | Seal-folk, aquatic                            |
| Shifter    | New          | Lycanthropic traits                           |

#### Class Modifications

Each folk may have class-specific modifications:

- Ability score substitutions (e.g., Halfling Bard: use LUK instead of CHA for spellcasting)
- Bonus skill points for certain class/folk combinations
- Special features unlocked by class/folk combination

**Reference:** [dnd.rigo.nu/folk](https://dnd.rigo.nu)

### 18+ Classes

Varlyn uses all standard D&D 5e classes with modifications, plus several homebrew additions:

#### Standard Classes (modified)

| Class     | Key Varlyn Modifications               |
| --------- | -------------------------------------- |
| Barbarian | Rage interacts with skill system       |
| Bard      | Inspiration uses skill checks          |
| Cleric    | Domain-based skill bonuses             |
| Druid     | Nature skill integration               |
| Fighter   | Origins system, weapon group mastery   |
| Monk      | Unarmored Defense via skill tree       |
| Paladin   | Oath affects skill selections          |
| Ranger    | Origins system, favored terrain skills |
| Rogue     | Expertise via skill purchases          |

| Warlock   | Pact affects skill tree access         |

| ------------ | ----------------------------------- |

| Cavalier     | Mounted combat specialist           |

### Weapon Groups

Standard simple/martial weapon proficiency is replaced with **12 weapon groups** at **4 ranks**:

| Group         | Type         |
| ------------- | ------------ |
| Axe           | Melee        |
| Club          | Melee        |
| Fencing Sword | Melee        |
| Flail         | Melee        |
| Knife         | Melee/Ranged |
| Polearm       | Melee        |
| Spear         | Melee/Ranged |
| Sword         | Melee        |
| Whip          | Melee        |
| Bow           | Ranged       |
| Crossbow      | Ranged       |
| Sling/Thrown  | Ranged       |

**Ranks:**

- Rank 1: Can use the weapon normally
- Rank 2: Access to martial techniques
- Rank 3: Weapon Master abilities (Cleave, Riposte, etc.)
- Rank 4: Advanced Master techniques

**Reference:** [dnd.rigo.nu/rules/weapons](https://dnd.rigo.nu)

### Armor Skill Tree

Armor proficiency is part of the **skill tree system**:

```
Light Armor
├── Unarmored Defense
└── Sneaky Armor
    └── Medium Armor
        ├── Shield Bearer
        └── Medium Armor Master
            └── Heavy Armor
                ├── Bulwark
                └── Mobile Guardian
```

- Must purchase lower tier before higher tier
- Uses skill points from the Practical tree
- Each tier unlocks armor types and special features
